# gh-actions-builder

A Simple easy-to-use github actions builder with many features - made with ai

Usage: [gh-actions-builder.pages.dev](https://gh-actions-builder.pages.dev)

# Features

## Workflow editing
- Live YAML generation with Monaco Editor
- Instant sync between UI and output

## Triggers
- push, pull_request, schedule, workflow_dispatch, release, workflow_call
- Supports branches, tags, paths, and event filters

## Permissions
- Granular GitHub token permissions
- read / write / none per scope

## Jobs
- Multiple jobs support
- Configurable runners (Ubuntu, Windows, macOS, custom)

## Steps
- Easy to use prebuilt templates (checkout, setup node/python, cache, git config, artifacts)
- Custom actions (uses) and shell steps (run)
- Editable step parameters
- Step builder
- Prebuilt + custom modes
